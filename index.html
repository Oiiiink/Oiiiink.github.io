链表的基本操作

  • createList(int n) // 创建一个具有n个结点的链表，这里没有写返回值
  • getLength(Nodeptr list) // 获得链表的长度
  • printList(Nodeptr list); // 打印输出一个链表的data域
  • Insert(Nodeptr list, ElemType elem)
  • searchNode(Nodeptr list, ElemType elem) // 在链表中查找某一数据元素
  • deleteNode(Nodeptr list, ElemType elem) // 在链表中删除包含某一数据元素的结点
    四大主要操作：创建、查找、插入、删除
